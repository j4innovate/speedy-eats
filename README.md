## ✨ Features & Highlights

- 🛵 **Offline Prices, Online Convenience** — Pay what you see on the physical restaurant menu with 0% markup.
- 💰 **Lowest Guaranteed Item Prices** — Delicious dishes starting at just ₹49, with most items under ₹250.
- ⚡ **Free Lightning-Fast Delivery** — Zero delivery fees on all food orders above ₹99.
- 🎨 **3D Interactive Avatars & Floating Badges** — Micro-animated, pop-out 3D avatars engineered with pixel-perfect responsive alignment.
- 📱 **Mobile-First Responsive Design** — Flawlessly optimized across Mobile, Tablet, and Desktop viewports.
- ⚡ **Zero Dependencies / Ultra Fast** — Light weight, hyper-optimized asset rendering with Tailwind CSS CDN.

## 💻 Local Setup & Development

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/j4innovate/speedy-eats.git
   cd speedy-eats
   ```

2. **Run a local development server:**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Or using Node npx serve
   npx serve .
   ```

3. **Open in browser:**
   Navigate to `http://localhost:8000` or `http://127.0.0.1:8000` in your web browser.
